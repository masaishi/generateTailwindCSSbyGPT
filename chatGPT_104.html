About Rust-Bert

Rust-Bert is a powerful NLP library that can perform a variety of tasks, including translation, summarization, question answering, and more. It supports multiple models and is capable of multi-threaded tokenization and GPU inference.

The library relies on the tch crate for bindings to the C++ Libtorch API and allows for automatic or manual installation of the required libraries.

Features

Multi-Task Learning

Rust-Bert features a wide variety of pre-trained models that are capable of performing multiple NLP tasks, including translation, summarization, and question answering, among others.

Multi-Threaded Tokenization

Rust-Bert is capable of performing multi-threaded tokenization, which significantly speeds up the processing of large amounts of text.

GPU Inference

Rust-Bert supports GPU inference, which allows for even faster processing of text data.

Easy Installation

Rust-Bert relies on the tch crate for bindings to the C++ Libtorch API and allows for automatic or manual installation of the required libraries.
