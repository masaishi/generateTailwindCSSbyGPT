Data-efficient Learning with SemDeDup

SemDeDup is a new method for data-efficient learning through semantic deduplication, which can remove semantically similar but not identical data pairs to make models learn faster with less data. With SemDeDup, you can remove 50% of the data with minimal performance loss, effectively halving training time. It also improves performance out of distribution and provides more efficiency gains compared to prior approaches.

Features and Benefits

Data-efficient Learning

SemDeDup removes semantically similar but not identical data pairs to make models learn faster with less data. It can remove 50% of the data with minimal performance loss, effectively halving training time.
