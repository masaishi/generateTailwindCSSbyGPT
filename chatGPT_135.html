Introduction to Yggdrasil Decision Forests

Yggdrasil Decision Forests (YDF) is a powerful library of algorithms that can be used for training, serving, and interpreting Decision Forest models. It is available in C++, CLI, TensorFlow, Go, and JavaScript (inference only) and can be used for modeling, training, and interpreting data.

YDF Explorer Features

  • Modeling, training, and interpreting data
  • Support for multiple programming languages
  • Easy-to-use API and documentation
  • Scalable and customizable
  • Integrations with popular data science tools

Integrations and Compatibility

  • Compatible with popular data science tools such as Python, R, and MATLAB
  • Integrates with popular machine learning frameworks like TensorFlow and PyTorch
  • Supports a wide range of file formats for input and output
