About pyGAM

pyGAM is a Python library that uses Generalized Additive Models (GAMs) to enable users to model non-linear relationships and additivity while maintaining interpretability. GAMs offer great flexibility in modeling complex datasets and are particularly useful for financial modeling, often revealing hidden trends and insights that simpler models might miss.

The key concepts involved in the pyGAM model are penalized B splines, which smooth the model and prevent overfitting, and the link function, which relates the expected value of the dependent variable to the linear predictors.
