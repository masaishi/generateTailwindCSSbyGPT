Fraud Detection for VoIP PBX Systems

What is SentryNotify?

SentryNotify is a managed security service that monitors your VoIP PBX system for fraud and security risks. We use SentryPeer, a tool that collects data on bad actors attempting to make phone calls, to alert you of potential security risks before any damage occurs.

How it works

  1. Sign up for SentryNotify and provide us with your VoIP PBX system information.
  2. We will set up SentryPeer to monitor your system and collect data on bad actors.
  3. If a potential security risk is detected, we will alert you through email or SMS.
  4. You can take preventive measures before any damage occurs.

Features

  • User-owned data
  • User-submitted data
  • Embedded Distributed Hash Table (DHT) node using OpenDHT
  • Peer-to-Peer sharing of collected actors and data through OpenDHT
  • Local data copy
  • WebHook for POSTing bad actor JSON to a central location
  • Analytics - opt-in
