Managed Security Service for VoIP PBX Systems

SentryNotify offers a managed security service that monitors the organization's VoIP PBX systems through SentryPeer. It alerts the business owner of potential security risks, allowing them to take preventive measures before any damage occurs.

Features of SentryNotify

User-owned data

SentryNotify allows users to own their collected data and decide if they want to share it with the community.

User-submitted data

Users can submit data on bad actors attempting to make phone calls, helping to build a more extensive network of fraud detection.

Embedded Distributed Hash Table (DHT) node using OpenDHT

SentryNotify uses OpenDHT to create a decentralized network of fraud detection and prevention.

Peer-to-Peer sharing of collected actors and data through OpenDHT

SentryNotify allows users to share their collected data with the community, helping to build a more extensive network of fraud detection.

Local data copy

SentryNotify allows users to keep a local copy of their collected data, ensuring that the information is always available and secure.

WebHook for POSTing bad actor JSON to a central location

SentryNotify offers a WebHook service that sends bad actor JSON to a central location, making it easier for users to access and analyze the data.

Analytics - opt-in

SentryNotify offers an optional analytics service that allows users to track and analyze their collected data, providing valuable insights into their organization's security risks.
