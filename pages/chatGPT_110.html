GLASU: A Communication-Eﬃcient Algorithm for Federated Learning with Vertically Distributed Graph Data

GLASU is a new algorithm for federated learning on graph data that significantly reduces communication overhead. It is designed to handle graphs with vertical data distribution, where each client owns a part of the node features and a different edge set.

Problem, Background, and Related Works

The problem considered in this work is vertical federated learning on graph data, where computing clients collectively train a model based on the partial features of the same set of samples they possess. The goal is to learn a model that predicts multiple labels for each node without revealing each client’s raw local data.

Graph Convolutional Network

The graph convolution network (GCN) is a typical example of the family of GNNs used in this work. It performs neighborhood aggregation in every feature transformation layer, such that the prediction of a graph node is based not only on the information of this node but also that of its neighbors.

GLASU Algorithm

GLASU is a communication-efficient algorithm that trains a graph neural network (GNN) through vertical federated learning. It adopts lazy aggregation and stale updates to skip communication in neighborhood aggregation and in model updates, respectively, greatly reducing communication while enjoying convergence guarantees.

Numerical Results

Extensive experiments on real-world datasets show that GLASU effectively trains a GNN that matches the accuracy of centralized training, while using only a fraction of the time due to communication saving.
