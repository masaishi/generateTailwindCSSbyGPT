About GLASU

GLASU - Communication-Efficient Algorithm for Federated Learning

GLASU proposes a model-splitting method and communication-efficient algorithm to train a graph neural network through Vertical Federated Learning (VFL), where each client owns a part of the node features and a different edge set.

Features

Model-Splitting Method

GLASU splits the model across the clients and the server, reducing communication overhead through lazy aggregation and stale updates, achieving convergence guarantees and matching the accuracy of centralized training.
