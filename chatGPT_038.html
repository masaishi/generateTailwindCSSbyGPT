What is SentryNotify?

SentryNotify is a managed security service that monitors your organization's VoIP PBX systems through SentryPeer. It alerts you of potential security risks, allowing you to take preventive measures before any damage occurs.

How does SentryNotify work?

SentryNotify collects data on bad actors attempting to make phone calls and saves the IP address they came from and number they tried to call. It alerts you of potential security risks through a WebHook for POSTing bad actor JSON to a central location, allowing you to take preventive measures before any damage occurs.

Features

  • User-owned data
  • User-submitted data
  • Embedded Distributed Hash Table (DHT) node using OpenDHT
  • Peer-to-Peer sharing of collected actors and data through OpenDHT
  • Local data copy
  • WebHook for POSTing bad actor JSON to a central location
  • Analytics - opt-in
