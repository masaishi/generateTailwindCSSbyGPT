Data-Efficient Learning Through Semantic Deduplication

SemDeDup is a new method for data-efficient learning through semantic deduplication, which can remove semantically similar but not identical data pairs to make models learn faster with less data.

Efficient Learning

SemDeDup can remove 50% of the data with minimal performance loss, effectively halving training time.

Improved Performance

It also improves performance out of distribution and provides more efficiency gains compared to prior approaches.

Pre-trained Embeddings

SemDeDup uses pre-trained embeddings to identify and remove semantic duplicates in web-scale datasets, thereby preserving performance and speeding up learning.

Frequently Asked Questions

What is SemDeDup?

SemDeDup is a new method for data-efficient learning through semantic deduplication, which can remove semantically similar but not identical data pairs to make models learn faster with less data.

How does SemDeDup work?

SemDeDup uses pre-trained embeddings to identify and remove semantic duplicates in web-scale datasets, thereby preserving performance and speeding up learning.

What are the benefits of using SemDeDup?

SemDeDup can remove 50% of the data with minimal performance loss, effectively halving training time. It also improves performance out of distribution and provides more efficiency gains compared to prior approaches.
